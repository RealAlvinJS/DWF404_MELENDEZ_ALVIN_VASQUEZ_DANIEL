]

Descripción: Obtiene los detalles de un medicamento específico por su ID.

Respuesta:

{
    "id": "1",
    "nombre": "Aspirina",
    "cantidad": 100,
    "precio": 12.50
}

Descripción: Crea un nuevo medicamento.

Datos de entrada:

{
    "nombre": "Paracetamol",
    "cantidad": 150,
    "precio": 5.00
}

Respuesta:

{
    "message": "Medicamento creado exitosamente"
}

Descripción: Actualiza un medicamento existente por su ID.

Datos de entrada:

{
    "nombre": "Paracetamol",
    "cantidad": 200,
    "precio": 6.00
}

Respuesta:

{
    "message": "Medicamento actualizado exitosamente"
}

Descripción: Elimina un medicamento por su ID.

Respuesta:

{
    "message": "Medicamento eliminado exitosamente"
}
